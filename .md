# 🧠 19. FINAL ARCHITECTURE

```text id="arch1"

↓
API Gateway
↓

↓
RabbitMQ Event Bus
↓

```


Rate limiting & DDoS хамгаалалт

Security
API Gateway дээр rate limit байгаа ч per-tenant, per-table, per-session гэсэн давхар хязгаарлалт дутуу. Guest session-аас spam order ирэхээс хамгаалах хэрэгтэй.

Audit log / Activity trail

Compliance
Хэн, хэзээ, юу өөрчилсөн гэсэн log байхгүй. Order cancel, menu price өөрчлөлт, employee action бүрийг бүртгэх шаардлагатай — ялангуяа санхүүгийн маргааны үед.

Optimistic locking / Concurrency

Order Service
Нэг ширээнд 2 зочин зэрэг order хийхэд race condition үүснэ. Order version/ETag эсвэл distributed lock механизм тодорхойлогдоогүй байна.

Data retention & GDPR/хувийн мэдээлэл

Legal
Phone number (loyalty), email (receipt) цуглуулж байгаа ч хадгалах хугацаа, устгах эрх, consent flow тодорхойлогдоогүй. Монголын хувийн мэдээллийн тухай хуулийг харгалзах хэрэгтэй.
